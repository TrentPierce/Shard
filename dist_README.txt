# Shard Node

Welcome to Shard! This folder contains everything you need to run a Shard node on your computer.

## Quick Start

### Option 1: Run Directly (Easiest)
Just double-click `ShardAI.exe` to start the node. It will run and you can access the web interface at http://localhost:3000

### Option 2: Install to Your Computer
1. Double-click `install.bat`
2. Choose option 1 for auto-start, or option 2 for manual install
3. The node will be installed to Program Files\ShardNode

## What Does Shard Do?

Shard is a decentralized AI inference network. When you run a node:
- You contribute your computer's idle compute power
- In return, you get priority access to AI inference
- The network verifies your contributions to ensure quality

## Requirements

- Windows 10 or Windows 11 (64-bit)
- 4GB+ RAM recommended
- Internet connection

## Running the Node

- **Web Interface**: http://localhost:3000
- **API**: http://localhost:8000
